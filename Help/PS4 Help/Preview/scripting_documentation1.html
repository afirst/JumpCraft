Scripting Documentation Up

 

 
This tutorial will guide you through the process of making a simple platform game using JumpCraft.  Follow the instructions in order below:

I. Creating a new game

  1. Click File, and select New Game Wizard.
  2. Click Next.
  3. Under Name of game, type Platform Game Tutorial.
  4. Under Credits where it says “Created by...,” replace MyName with your name.
  5. Under Company, type in the name of your company or leave it blank.
  6. Under Website, type in the name of your website, including the “http://”.
  7. Under Support email, type in your support email address, or your personal email address.
  8. Click Next, and then click Next again.
  9. Make sure the Windowed option is checked, and make sure the window size is set to 450x300.  Under Window title, type Platform Game Tutorial.
  10. Click Next.
  11. Choose a color scheme (optional), and click Next.
  12. Click Next, and then click Finish.

II. Setting up a tileset

  1. Click the Tools menu, and select Tileset Editor.
  2. Click Clear.  When it asks if you are sure, click Yes.
  3. To set the default transparent color for the tiles we are about to add, click the ... button under transparent color.  Select Magenta, or the second box from the top on the right column.  Then click OK.
  4. Now we are going to add the tiles to the tileset.  Click Add.  Navigate to the installation folder (normally C:\Program Files\JumpCraft [version number]. Browse to the Graphics folder, and then the Tileset folder.  Select all of the files in the current view by dragging a box around then or pressing Ctrl+A. Click Open.
  5. Now, we must configure the tiles in the tileset (their size, animations, and behavior). Each tile that needs configuration will be shown in italics and the instructions will follow under that. (Reminder: For all tiles below, select them in the tile list before editing them).
    • Bullet
      Under Section W, select 8.
      Under Behavior, click (...).  Scroll up and select Bullet.  Click OK.
      Under Animations, click Edit.  Under the animation dropdown, select Disappear.  Change the Start Frame to 2 and the Stop Frame to 4.  Under Frame Interval, type 0.04.  Click OK.
    • Character
      Under Section W, select 64.
      Under Section H, select 64.
      Under Behavior, select Character.
      Under Animations, click Edit.  Under the animation dropdown, select Walk Right.  Change the Start Frame to 1 and the Stop Frame to 10.  Under Frame Interval, type 0.15.  Under the animation dropdown, select Walk Left.  Change the Start Frame to 11 and the Stop Frame to 20.  Under Frame Interval, type 0.15.  Under the animation dropdown, select Climb.  Change the Start Frame to 21 and the Stop Frame to 26.  Under Frame Interval, type 0.15.  Click OK.
    • Clouds
      Under Behavior, select Background.
    • Coin
      Under Behavior, click (...).  Scroll up and select Coin.  Under Sound on collision, click (...).  Under Sound file, click (...).  Navigate to the installation folder, browse to the Sounds folder, and double-click Coin.wav.  Under Pan, enter 44.  Click OK.  Click OK again.
      Under Animations, click Edit.  Under the Normal animation, set the Start Frame to 1, the Stop Frame to 10, and the Frame Interval to 0.1.  Under the Disappear animation, set the Start Frame to 11, the Stop Frame to 14, and the Frame Interval to 0.1.  Click OK.
    • Enemy Shooter
      Under Behavior, select Background.  (Note: We will set the behavior for this tile later using the Action Editor).
      Under Animations, click Edit.  Under the Normal animation, set the Start Frame to 1, the Stop Frame to 2, and the Frame Interval to 1.  Click OK.
    • Exit
      Under Section W, select 64.
      Under Section H, select 96.
      Under Behavior, click (...).  Select Exit and click OK.
    • Grass
      Under Behavior, select Background.
    • Guides
      Under Behavior, select Background.
    • Heart
      Under Behavior, click (...).  Scroll up and select Heart.  Under Sound on collision, click (...).  Under Sound file, click (...).  Navigate to the installation folder, browse to the Sounds folder, and double-click Heart.wav.  Click OK.  Click OK again.
      Under Animations, click Edit.  Under the Normal animation, set the Start Frame to 1, the Stop Frame to 6, and the Frame Interval to 0.1.  Under the Disappear animation, set the Start Frame to 9, the Stop Frame to 12, and the Frame Interval to 0.1.  Click OK.
    • Ice
      Under Behavior, click (...).  Select Ice.  Under Slip speed, enter 3.  Click OK.
    • Ladder
      Under Behavior, click (...).  Scroll up and select Ladder.  Click OK.
    • Ledges
      Under Behavior, select Ledge.
    • Spike
      Under Behavior, click (...).  Select Spike Up.  Click OK.  (Note: We will set the decrease health sound later in the Action Editor.)
    • Tree
      Under Behavior, select Background.
    • Unstable
      Under Section H, select 16.
      Under Behavior, click (...).  Select Disintegrate.  Click OK.
      Under Animations, click Edit.  Under the Disappear animation, set the Start Frame to 2, the Stop Frame to 8, and the Frame Interval to 0.07. Click OK.
    • Water
      Under Behavior, click (...).  Scroll up and select Water.  Click OK.
  6. Click Done, or if you want to make this tileset the default, click Set as Default.

II. Action Editor

  1. Click Tools and select Action Editor.
  2. We are going to make a new timer for the shooting enemy. Click the Timers button on the toolbar (the button that looks like a clock). Click Add.  Under Name, type Shoot Timer.  Under Interval, type 1.

    Click OK.
  3. Now we are going to make a new action for the Shoot Timer. Under the Triggers portion of the Action Editor, double-click Timers, double-click Shoot Timer, and click Tick.  On the toolbar on the upper-right portion of the Action Editor, click New Action (the leftmost button).  Select the Map tab.  Under Action type, select Shoot.  Now look under Parameters.  Double-click the first item (Bullet generator) and select Enemy Shooter in the dropdown box.  Click OK.  Double-click the next item (Bullet tile).  Select Bullet and click OK.  Under the next item (Direction), select Right, and under Speed, enter 5.

    Click OK.
  4. Our next action will be to add the decrease health sound effect.  Under the Triggers portion of the Action Editor, double-click Counters, double-click Health, and click Value Changed.  On the actions list, select the If Current Counter Decreased action.

    On the toolbar click New Action.  Select the Misc. (Miscellaneous) tab, and choose Play Sound.  Under Parameters, and double-click Sound, click (...).  Navigate to the Sounds folder from the installation folder, and double-click Lose Health.wav.  Click OK, and click OK again.
  5. We are done with the Action Editor, so click OK.

III. Window Editor

  1. Click Tools and select Window Editor.
  2. We are going to create some different types of counters in the game window. In the Windows panel on the right side of the Window Editor, select Game.
    Delete the health counter (3    rd control from the top).  To delete a control, select it by clicking it with the mouse and press the delete key on your keyboard. In the Controls panel on the left side of the Window Editor, select Image & Text Counter.  Where the health counter used to be, drag a new Image & Text Counter in its place.  Double-click the new control. Under Counter, select Coins.  Check Shadow.  Under File, click the (...) button. Navigate to the installation folder, browse to the Graphics folder, then Counter Icons folder, and double-click coins.bmp.  Under Transparent Color, click (...) and select Magenta (second box from the top on the right column). Click OK to close the color selectioin dialog. Under Width, enter 24, and under Height, enter 24.

    Click OK.

    Now we are going to create the health counter. Select the Image Counter control, and place a new one under the coin counter.  Double-click it.  Under Counter, select Health.  Under File, click (...), and browse to the installation folder, then go to Graphics, and then Counter Icons.  Double-click hearts.bmp.  Choose White as the transparent color (bottom-right box under Basic Colors in the color selector dialog).  Under Padding, enter 0.

    Click OK.

    Deselect the health counter by clicking Edit and choosing Deselect.
  3. The last thing we are going to do in the window editor is to create a High Score window.  In the Windows panel on the right side of the Window Editor, select the Help window so we can use it as a template.  On the toolbar below the list of windows, click the Copy Window button (second button from the left). Then click the Rename Window button (third button from the left). Under Window Name, click the dropdown arrow and select High Scores.

    Click OK.
    Double-click the Help label and change the text to High Scores, and then click OK.  Delete the second label down (in the middle of the window).  In its place, add a High Scores Area control.  Move the top label upwards, move the OK button downwards, and resize the High Scores Area control so that there is more room to display high scores.
  4. We are done with the window editor, so close it by clicking the X button on the top-right corner. (Note: In the window editor and ONLY in the window editor, the X button acts like an OK button. In any other dialog, the X button will function as a Cancel button.)

IV. Level Editor

  1. We should already be in the level editor, which is always the top-level window. Your screen should look something like this:
  2. Notice that we are currently working in the Foreground layer by looking in the Layers panel on the right side of the Level Editor. To draw on a different layer, select the “Active” checkbox for the layer you want to draw on. You can change the state of the “Visible” checkboxes to show or hide layers. For now, leave the Foreground layer active.
  3. To use the Tiles panel on the left side of the Level Editor, select the tile you want to draw with, and, if a subtile selector is shown below the tiles list, click the subtile you want to draw with. When you select a tile, you switch to Draw mode (shown at the toolbar on the top of the Level Editor). If you click in the white area to the right of the tiles, your tile is deselected and you switch to Erase mode. Right now, select the Blocks tile and the top-left subtile.  This is shown in the screenshot in step 1.
  4. Now, begin drawing with the selected tile. Make it look like the screen shown below:
  5. Select the top-right subtile and draw with it to make your level look like this:
  6. Now, draw with some other tiles to make your level look like this:
  7. For this level, we want the block above the spikes to move back and forth from one end of the spikes to the other. To do this, we need to set a path for it.  On the toolbar at the top, click Select.  Then click the block tile.  Notice that the Properties panel on the right side of the editor is enabled, so you can modify the tile.  Under Path, click Edit.

    Notice how the interface changes.  The Tiles panel is replaced with the Toolbox, the Layers and Levels panels are now gone, and the Properties panel is replaced with the Path Properties panel. Also, the background dims and the grid moves to the centers of the tiles instead of bordering them. Create a new point for the path by clicking in the center of the selected tile. Then, make another point by clicking in the spot a tile under the rightmost coin tile. Your screen should look like this:

    For this path, we do not need to modify the speed; the 3-second speed will work just fine.  Click Preview to preview how your tile will move before you aceept the changes. Then click OK.
  8. The last thing we need to do with this test level is to resize it. To do this, hover your mouse over the bottom-right tile that you drew, and look at the coordinates on the status bar (at the bottom of the editor). The X coordinates should be 12 and the Y coordinate should be 14. Click Edit and select Resize Level.  Under Width, type 12, and under Height, type 14.

V. Testing

  1. First, save your project if you wish to. Click File, and choose Save As.  Type in a name for your file and click Save.
  2. To test your game, click Run and choose Start, or, click the blue “play”-shaped button on the toolbar.  It may take several seconds to compile. When it’s done, a window like this should open:
  3. Click Play.
  4. Congratulations, you have just created a platform-based game!
